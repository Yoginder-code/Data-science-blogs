Why we using numpy?

Answer : We using numpy cause of following reasons :
1. The biggest advantage of numpy is that its very fast cause numpy is written in C.
2. Numpy is fast because its based upon the homogenity of data unlike python where data is stored in heterogenous manner.
3. It requires very less code as compare to python.


Let's understand why numpy ?

Example : The following example shows how numpy require very less amount of code :-

What is Net Promoter Score (NPS)?



Answer : Net Promoter Score (NPS) is a customer loyalty and satisfaction measurement taken from asking customers how likely they are to recommend your product or service to others on a scale of 0-10.

Example : Suppose you are data analyst at Airbnb and you need to analyze the user survey data and report its NPS to the management. Here we will try to solve this business problem using python library called numpy, by just few lines of code we are able to draw wonderful insights.

About the dataset :
1. We have Airbnb dataset which contains score given by customers from 0 to 10.
2. Scores from 0 to 6 are called detractors which means that customer is very unlikely to recommend Airbnb services to others.
3. Scores of 9 to 10 are called promoters which means that customer is highly likely to commend Airbnb services to others.
4. Scores of 7 to 8 are called passive and they are like neutral customers.

Note: Net promoter score (NPS) = Percentage of Promoters / Percentage of Detractors.

Dataset link


Let's calculate Airbnb NPS :- 

                        
# loading the dataset:
score = np.loadtxt("..file location..",dtype="int")
total = score.shape[0]
print('Total scores we have : ',total)
output:
----------------------------------------------------------------------------------------------------
Total scores we have :  1167
----------------------------------------------------------------------------------------------------
                        
                    
                    
#extracting detractors from score : 
detractor_mask = score<7
detractor = score[detractor_mask]
detractor_no = detractor.shape[0]
percentage_detractor = detractor_no*100/total
print('The percentage of detractors are : ',round(percentage_detractor),'%')
output:
----------------------------------------------------------------------------------------------------
The percentage of detractors are :  28 %
----------------------------------------------------------------------------------------------------
                    
                   
                    
promoter_mask = score>8
promoter = score[promoter_mask]
promoter_no = promoter.shape[0]
percentage_promoter = promoter_no*100/total
print('The percentage of promoters are : ',round(percentage_promoter),'%')
output:
----------------------------------------------------------------------------------------------------
The percentage of promoters are :  52 %
----------------------------------------------------------------------------------------------------

                    
                
                    
nps = percentage_promoter-percentage_detractor
print('The Net Promoter Score of Airbnb : ',round(nps),'%')
----------------------------------------------------------------------------------------------------
The Net Promoter Score of Airbnb :  24 %
----------------------------------------------------------------------------------------------------

Observation :


1. The NPS of Airbnb is 24%. It help us to know to know how many of its customers were assets and how many were liabilities. By correlating the customer's subjective response to an objective number, the metric can be used to legitimately drive a company's internal priorities.

2. Just by few lines of python code we can easily find out NPS of any company and this metric gives us useful insight related to customers.

3. We can also observe how numpy library of python is effective and effecient way to analyse the data.
